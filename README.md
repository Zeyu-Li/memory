
# Memory

## About

This program is a matching game that consists of a 4x4 matrix.
You try to match the tiles of two chooses tiles by flipping the tiles

## How to Use

To use this program simply:

For MacOS or Linux:
    python3 memory_v3.py

For Windows:
    python memory_v3.py
